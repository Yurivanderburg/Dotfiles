# Dotfiles

My i3 setup with with the .config files 


1. Install basic depencies:
```
apt install git i3 polybar alacritty zsh picom rofi xdotool feh brightnessctl xbacklight
```

2. Install oh-my-zsh and p10k manually, then replace the config files. Set ```ZSH_THEME="powerlevel10k/powerlevel10k"``` in ```~/.zshrc```. Note that zsh-autosuggestions must be installed separately.

3. 



Setup:

- OS: [TuxedoOS](https://github.com/tuxedocomputers)
- WM: [i3-wm](https://i3wm.org/)
- Bar: [polybar](https://github.com/polybar/polybar)
- Terminal: [Alacritty](https://github.com/alacritty/alacritty)
- Shell. [ZSH](https://www.zsh.org/) and [Oh My Zsh](https://github.com/ohmyzsh/ohmyzsh)
  - Plugins:
    - dirhistory
    - git
    - history
    - [Powerlevel10k](https://github.com/romkatv/powerlevel10k)
    - sudo
    - ubuntu
    - fastfile
    - virtualenv
    - web-search
    - zsh-autosuggestions
- Font: [JetBrainsMono Nerd Font](https://www.nerdfonts.com/)
- Compositor: [Picom](https://github.com/yshui/picom)
- App Launcher: [rofi](https://github.com/davatorium/rofi)
- Theme: [Breeze](https://github.com/KDE/breeze)
- Notifications: [Dunst](https://github.com/dunst-project/dunst)
